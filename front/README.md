## WARNING

package.json 안의 패키지들 꼭 설치해줍시다.

```
yarn install
```

## BUILD

build 방법

```
yarn build -p
```

~path/build 경로에 빌드 파일 생성됩니다.

nginx 설정에서 ~path/build/index.html 을 경로로 설정해주면 됩니다.
